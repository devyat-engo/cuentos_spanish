################# What it does? #################
Standalone
The program has a basic menu with which the user can interact
and retrieve different fairytales sentence by sentence
so the user can calmly read and understand each sentence
without the burden of a complete story in one shot.

Slack
The bot must be alive so it can listen to the user.
Only difference with the standalone app is that
the user cannot choose which new fairytale to read,
a new one can be chosen but will be random.

* More Fairytales can be added to the database in .txt format
#################################################
